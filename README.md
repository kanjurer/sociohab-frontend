# sociohab


Stage 1:
  Functionalities:
    1) Habit Tracker
      a) Track progress using charts
      b) Set reminders
      c) Set goals
      d) Track goal progress by checking in daily targets
    2) Groups which will be assigned habits
    3) Compete with other people
    4) Post/Social
    5) Direct Messages
  UI:
    1) Responsive
    2) Dark Mode
  Implementation:
    1) JS
    2) ReactJS
    3) User Authentication and Authorisation (JWT, Passport, etc.)
    4) NodeJS
    5) ExpressJS
    6) MongoDB
    7) NextJS
    8) Socket.io
    9) TS
